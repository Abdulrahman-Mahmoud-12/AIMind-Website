# Inside the Mind of AI 🧠

## Overview
An interactive futuristic website that explains Artificial Intelligence
in a visually engaging way. The experience feels like exploring
the brain of an AI system, combining rich visuals, video, animations,
and interactivity built entirely with HTML CSS and JavaScript.

## Live Demo
> https://username.github.io/ai-mind-website

## Features
- 🎬 AI-generated background video in hero section
- ✍️ Animated typing text effect cycling through AI quotes
- 🌌 Interactive floating particle system with mouse interaction
- 🧠 Clickable neural network visualization with animation
- 🎥 Four video demo modals with internal audio
- 🎮 Rock Paper Scissors game vs AI with score tracking
- 📜 Scroll-reveal animated AI history timeline
- 💫 Animated hologram rings in final message section
- 📱 Fully responsive for mobile tablet and desktop
- 🌙 Dark futuristic UI with glassmorphism and neon glow

## Tech Stack
- HTML5 (semantic structure)
- CSS3 (animations glassmorphism custom properties)
- Vanilla JavaScript (classes OOP canvas API)
- Font Awesome 6.5 (icons)
- Google Fonts: Orbitron, Poppins, Exo 2

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Abdulrahman-Mahmoud-12/AIMind-Website.git
   ```

2. Navigate into the folder:
   ```bash
   cd ai-mind-website
   ```

3. Add your asset files:
   Place all images inside assets/images/
   Place all videos inside assets/videos/
   Use exact filenames listed in Project Structure above

4. Open in browser:
   Simply open index.html in any modern browser
   OR use VS Code with Live Server extension for best experience

5. Recommended browser:
   Google Chrome or Microsoft Edge for best video support
